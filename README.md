# 📦 Sistema de Gerenciamento de Almoxarifado de TI

## 📝 Descrição da Aplicação
Este projeto é um sistema desenvolvido para gerenciar o acervo de itens de hardware (como laptops, projetores e kits de Arduino) de um laboratório acadêmico. O objetivo principal é automatizar e controlar o fluxo de empréstimos, garantindo rastreabilidade e evitando conflitos de alocação de recursos. 

A aplicação lida com dois perfis de usuários:
* **Técnico (Admin):** Responsável por gerenciar o inventário, cadastrar equipamentos e monitorar as movimentações.
* **Aluno:** Usuário final que realiza a retirada e devolução dos materiais acadêmicos.

A lógica central do sistema baseia-se na vinculação de um Aluno a um ou mais Equipamentos, com travas de segurança rigorosas: o sistema bloqueia automaticamente qualquer tentativa de empréstimo de itens que constem com o status "Em Manutenção" ou "Em Uso".
